# Faq Wordpress Plugin
- Faq plugin for wordpress blog post where it serve the faqs in accordions just after the the_content() and faq schema for the same blog post.

## plugin details
- Plugin name: Faq
- Plugin slug: faq
- Plugin version: 1.0.0
- Plugin author: Tech Dev 247
- Plugin author url: https://techdev247.com/
- Plugin description: Faq plugin for wordpress blog post where it serve the faqs in accordions just after the the_content() and faq schema for the same blog post.

## Here are the high level features of this plugin
- In the admin side there will be a custom meta box where user can add the faqs.
- For adding the faqs user can add the faq title and faq content.
- We will use the repeatable fields for adding the faqs. where admin can add and remove the faqs.
- In the front end we will show the faqs in accordions just after the the_content().
- We will add the faq schema for the blog post at the front end side by using these faqs also provide a action and filter hooks to modify the faqs schema.
- One more thing we will add a menu in the admin to set the execution of this plugin like just after the_content() or shortcode.
- We will provide a functionality if developer want to override the output of the faqs.
